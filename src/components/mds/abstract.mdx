# Abstract

随着深度神经网络的日益普及，后门攻击的日益盛行对其安全性构成了重大威胁。现有的后门攻击大多为脏标签攻击，很容易通过人工检测发现，而干净标签攻击具有更强的隐蔽性。然而，在严格约束条件下进行有效的干净标签攻击仍然是一个重大挑战。本文发现数据投毒过程中样本的选择策略对干净标签攻击有效性的影响至关重要。本文引入了一种新型干净标签后门攻击框架，该框架利用特征对齐来加强源标签样本与目标标签之间的关联，通过计算目标标签样本的特征质心，并将源标签样本的特征与这些质心对齐，显著提升了攻击成功率。在多个基准数据集上进行的一系列实验表明，本文所提出的方法在严格的约束条件下（低中毒率和有限的扰动预算）表现优于现有的干净标签后门攻击，同时在现实场景中对良性样本保持了较高的准确性。此外，本文中所提出的方法对后门防御有较好的鲁棒性，凸显了其潜在威胁。
